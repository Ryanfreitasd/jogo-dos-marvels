## Tecnologies

- Angular 17.3.2
- SCSS
- Typescript
- HTML5

# Instalation

- Clone the repository

```bash
git clone https://github.com/Ryanfreitasd/jogo-dos-marvels.git
```

- Run `npm install`
- Run `ng serve`
